# Prologue Combat System Documentation

## Overview

프롤로그 전투는 `FlowStatePrologueCombat` 클래스에서 관리되며, 11단계의 스크립티드 시나리오로 구성됩니다.
다이얼로그와 전투 액션이 번갈아가며 진행되어 스토리를 전달합니다.

## 시나리오 흐름

### 초기화 단계
1. 카메라 줌인 (Size: 5.0 → 7.0)
2. 보드 색상 회색으로 변경
3. 모든 캐릭터 Y축 +4 방향으로 이동 (2초)
4. 모든 캐릭터 정지

## 단계별 상세 액션

### 1단계: WitchAttackPrepare (마녀의 선제공격)
```
- 마녀에게 공격 준비 이펙트 부착 (fx_common_synergy_lightning_01)
- 1.5초 대기
```

### 2단계: ClaySkillAndWitchAttack (클레이의 방어)
```
- 클레이 Y축 +1 이동 (중앙으로)
- 이동 완료 됐을 때 클레이 스킬 발동
- 마녀 공격 준비 이펙트 제거
- 마녀 스킬 발동
- 1.5초 대기
```

### 3단계: ClayGroggy (클레이의 맹세)
```
- [TODO] 클레이 그로기 모션 재생
- 1초 대기
```

### 4단계: FreeAttack3Seconds (유니 & 필리아의 반격)
```
- 아트레시아, 유니, 필리아 → 마녀 공격 (Idle 상태)
- 마녀도 평타 공격
- 3초 자유 전투
- 마녀 지친 모션 (아직 state 개발 안됨.)
- 모든 캐릭터 정지
```

### 5단계: WitchAoEAndCharactersDown (마녀의 역습)
```
- 마녀 지친 모션 -> 아이들로 변경
- 0.5초 대기
- 마녀 광역 스킬 발동
- 2초 대기
- 유니, 필리아, 클레이 → Dead 상태
- 1초 대기
```

### 6단계: MarieJoin (마리에 합류)
```
- 마리에 소환 (위치: 2, 7 또는 빈 타일)
- 마리에 타겟 → 마녀
- 마리에 스킬 발동
- 1.5초 대기
```

### 7단계: MarieDown (마리에의 희생)
```
- 아트레시아 0.5초 대기 후 마녀 공격
- 마리에 마녀 공격
- 마녀 → 마리에 공격
- 1초 대기
- 마리에 → Dead 상태
- 마녀 그로기로 변경 (state 아직 개발 안됨)
- 1초 대기
```

### 8단계: ArtesiaSupernovaAndSkill (아트레시아의 각성)
```
- [TODO] 아트레시아 초신성 모드 활성화
- 아트레시아 스킬 발동
- 1.5초 대기
```

### 9단계: WitchHpRecoverAndFinalPrepare (마녀 체력 회복)
```
- [TODO] 마녀 체력 회복 처리
- [TODO] 마녀 최후 스킬 대기 모션
- 1.5초 대기
```

### 10단계: WitchFinalPrepareFx (마녀의 진정한 힘)
```
- 마녀 최후 공격 준비 이펙트 (fx_common_synergy_lightning_01)
- 1초 대기
```

### 11단계: WitchFinalAttackAndArtesiaDefend (최후의 순간)
```
- 마녀 최후 공격 준비 이펙트 제거
```
